🎯 这是什么项目

Kite 是一个面向的桌面 AI 助手，将 Claude Code SDK 包装成图形界面。用户无需命令行或编程知识即可使用 AI 完成：

  • 原型设计 — 自然语言描述，AI 生成代码、创建文件、实时预览
  • 数据分析 — 上传文件，AI 自动处理并生成可视化报告
  • 内容创作 — 翻译、SEO 优化、多轮对话迭代
  • 自动化任务 — 创建自定义 Skills，一键执行重复操作

🛠️ 技术栈

Electron React 18 TypeScript Zustand Tailwind CSS @anthropic-ai/claude-code MCP (Model Context Protocol)

🏗️ 重构前后对比

重构前 (ai-browser)

  • 自实现完整浏览器自动化
  • CDP 协议封装 (navigation, input, network, performance, console...)
  • SDK MCP Server 内置
  • snapshot / emulation 工具集
  • ~5,000 行代码

重构后 (Chrome Debug Launcher)

  • 复用 Chrome DevTools Protocol
  • 通过 MCP 扩展模式连接
  • 更轻量的 launcher 逻辑
  • 职责分离：launcher 只负责启动和连接
  • 12,057 行（但更内聚）
